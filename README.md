# How to use
```bash
git clone https://github.com/vitosotdihaet/map-pinner
cd map-pinner
mv .env-example .env
docker compose up --detach
```

You can change server's properties by editing the .env file. They will be changed on the next start of the server
```bash
edit .env
```

# Start
```bash
docker compose up --detach
```

# Stop
```bash
docker compose down
```
